# 📸 Bootstrap Photography Page

A responsive webpage built using **Bootstrap 5**, featuring a navigation bar, a photography card section, and content layout.

---


---

## ✨ Features

- ✅ Responsive **Navbar** with Home, Features, Pricing, and Disabled links
- ✅ **Search bar** integrated in the navbar
- ✅ **Photography Card** with image, description, and a "Know More" button
- ✅ Clean content/text section below the card
- ✅ Built with **Bootstrap** — fully responsive on all screen sizes

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure |
| CSS3 | Custom styling |
| Bootstrap 5 | Responsive layout & components |

---

## 📁 Project Structure

```
bootsrap.html

├── Photography Card Section

```
